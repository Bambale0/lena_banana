# User Flow

## User

1. `/start`
2. Select `Nano Banana` or `Motion Control`
3. Select model
4. Upload reference files when required
5. Enter prompt and quality options
6. Bot creates KIE task and deducts credits
7. Background tracker sends results when ready
8. User can view balance, buy packages, open gallery, or open partner links

## Admin

1. `/admin`
2. Open management section
3. Adjust models, packages, payments, users, gallery, partners, settings, or broadcast text
4. Changes are written to Postgres and reflected in bot menus
